## Features

- Responsive Design ensures seamless and optimal user experience across various devices, adpating to different screen size.
- Responsive Explore section with images and hoover effect.
- Contact form and success page.
- Did You Know section with multimedia elements, such as videos some visual effects.

## Existing Features

### Nav Bar

- Featured on all three pages, the full responsive nagivation bar includes links to the Logo, Home page, Explore, and Contact.
- The navigation bar is identical on each page, ensuring consistent and easy navigation.
 This design choice enables users to seamlessly move between pages on various devices without relying on the 'back' button.

## Deployment

- The site was deployed to GitHub pages. The steps to deploy are as follows: 
  - In the GitHub repository, navigate to the Settings tab 
  - From the source section drop-down menu, select the Master Branch
  - Once the master branch has been selected, the page will be automatically refreshed with a detailed ribbon display to indicate the successful deployment. 

## Installation Instructions

1. Clone the repository:

```bash
git clone https://milosson.github.io/MotherEarth/
```

2. Open the project in your preffered code editor.
3. Launch the website by opening the 'index.html' file in a web browser.



** The live link can be found here - <https://milosson.github.io/MotherEarth/index.html>
